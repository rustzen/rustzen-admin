# Documentation Governance Task

1. Check all documentation references and fix broken or outdated paths.
2. Check all documentation for duplicated content and consolidate each rule or fact into one owning file.
3. Check that every document has a clear responsibility and gives AI agents enough context to work correctly.
4. Ensure AI agents can find the right document before modifying, editing, or adding files, and can follow the documented boundaries without drifting from repository conventions and best practices.
5. Write documentation in English unless a task explicitly requires another language.
